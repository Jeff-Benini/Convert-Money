<h2 id="uso">Como Usar</h2>
<h2>
<ol>
  <li>Escolha a moeda que deseja converter.</li>
  <li>Escolha para qual moeda deseja converter.</li>
  <li>Aperte o botão converter.</li>
  <li>O Valor a ser convertido aparecerá com sua respectiva bandeira.</li>
  <li>O Valor que foi convertido aparecerá com sua respectiva bandeira.</li>
</ol>
